# Реєстр ЕСКО контрактів

Веб-застосунок для реєстрації та обліку договорів енергосервісу (ЕСКО контрактів). 

## Функціонал
- **Реєстрація нових договорів:** форма з валідацією полів (номер, замовник, об'єкт, вартість, економія, терміни).
- **Відображення списку:** асинхронне завантаження всіх зареєстрованих контрактів без перезавантаження сторінки.
- **Видалення:** можливість видалення неактуальних контрактів із підтвердженням дії.
- **Збереження даних:** усі записи зберігаються локально на сервері у форматі JSON.

## Технології
- **Backend:** Node.js, Express.js
- **Frontend:** HTML5, CSS3, Vanilla JavaScript (Fetch API)
- **База даних:** файлова система (JSON)
